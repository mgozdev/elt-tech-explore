# Exploration Notebooks

## Purpose

This directory contains Jupyter notebooks for interactive exploration and prototyping.

## Notebook Categories

### 1. Data Exploration
- Understanding N-PORT schema
- Data quality assessment
- Statistical analysis

### 2. Pattern Prototypes
- Quick implementations of each pattern
- Comparative analysis
- Performance testing

### 3. Use Case Examples
- Portfolio analysis examples
- Risk metrics calculation
- Time-series analysis

## Setup

```bash
pip install jupyter pandas polars duckdb matplotlib seaborn
jupyter notebook
```
